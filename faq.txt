Инструкция по запуску:
1. Установить nodejs >= 12.20.1 версии
2. Установить модули командой: npm install
3. Установить mysql-сервер >= 5.7.15 версии (если уже есть, то пропускаем этот шаг)
4. Создать базу данных c любым названием
5. Заполинть config/config.json, где:
  > token - Токен доступа для бота (Строка),
  > database.host: хост базы данных (если mysql на вашем пк, то localhost)(Строка),
  > database.port: порт базы данных (обычно 3306)(Число),
  > database.username: имя пользователя (обычно root )(Строка),
  > database.password: пароль пользователя (обычно пустой, если на локальном пк)(Строка),
  > database.database: имя базы данных, которую создали в шаге №4 (Строка)
  > logger.warns: Показывать предупреждения или нет (Логическое true \ false)
  > logger.errors: Показывать ошибки или нет (Логическое true \ false)
  > logger.logs: Показывать логи или нет (Логическое true \ false)
  > logger.debugs:Показывать сообщения отладки или нет (Логическое true \ false)
  > change_photo_cooldown_ms: Через сколько можно менять фото (Число в миллисекундах, дефолт: 6 часов)
  > admins: id админов (Массив чисел)
6. Запустить бота, командой npm run start:prod

P.S ** Если код будет изменён не изначальным разработчиком и что-то перестанет работать - фикс этого будет платный

** - Кроме текстовых переменных - их менять можно. (Пример такой переменной: let text = `Привет, ${user.nickname}`; )
